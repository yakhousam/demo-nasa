# Demo Nasa
## Description:
Lists data (taken from one of [NASA's open APIs](https://api.nasa.gov/)) concerning potentially hazardous space objects that threaten our existence on Earth! 
<br/>
Guides: [Understanding client-side web development tools (Mozilla)](https://developer.mozilla.org/en-US/docs/Learn/Tools_and_testing/Understanding_client-side_tools)
## Learned:
- [Parcel (Bundler)](https://parceljs.org/)
- [Netlify (Deploy)](https://www.netlify.com/)

```
npm install
```
